
# begin

**Description** : 
- Returns an iterator pointing to the first element in the vector.
				
**Example**:
```cpp
	// Iterate over myVector using iterators
	for (auto it = myVector.begin(); it != myVector.end(); it++) {
	    std::cout << *it << " ";
	}
```
