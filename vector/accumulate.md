# accumulate

**Description** : 
- accumulates elements in the specified range
- takes elements in order [first,last)

**Example**:
```cpp
	//Declare the vector
    std::vector<int> num{10,15,20};
    //Function to accu values to the vector
    cout << accumulate(num.begin(),num.end(),0);
```
